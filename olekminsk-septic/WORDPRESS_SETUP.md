# Инструкция: Установка лендинга на WordPress

## Вариант 1: Простой (HTML-страница в WordPress)

### Шаг 1: Установка плагина
1. Зайдите в админку WordPress
2. Плагины → Добавить новый
3. Найдите "Insert Headers and Footers" или "Custom CSS JS"
4. Установите и активируйте

### Шаг 2: Создание страницы
1. Страницы → Добавить новую
2. Название: "Откачка септиков Олекминск"
3. В редакторе выберите "Редактор кода" (Code Editor)
4. Вставьте содержимое index.html
5. Опубликуйте

### Шаг 3: Добавление стилей
1. Внешний вид → Дополнительные стили (Custom CSS)
2. Вставьте содержимое css/style.css
3. Сохраните

### Шаг 4: Добавление скриптов
1. Внешний вид → Редактор темы
2. Откройте footer.php
3. Перед закрывающим `</body>` вставьте:
```html
<script>
// Вставьте содержимое js/main.js
</script>
```

## Вариант 2: Через плагин (Рекомендуется)

### Используйте плагин "Elementor" или "WPBakery"

1. Установите Elementor (бесплатно)
2. Создайте новую страницу
3. Нажмите "Редактировать в Elementor"
4. Вставьте блоки по структуре лендинга:
   - Шапка с телефоном
   - Hero section с заголовком
   - Сетка услуг (4 карточки)
   - Преимущества (6 блоков)
   - Таблица цен
   - Этапы работы
   - Форма заявки
   - Контакты

### Цвета для настройки:
- Основной синий: `#1e3c72` или `#2a5298`
- Акцентный оранжевый: `#ff6b35`
- Фон: `#f8f9fa`
- Текст: `#333333`

## Вариант 3: Готовая тема-лендинг

### Установите тему:
1. Внешний вид → Темы → Добавить новую
2. Найдите: "Astra", "Neve" или "OceanWP"
3. Установите и активируйте
4. Импортируйте готовый шаблон лендинга
5. Отредактируйте под свои нужды

## Важно для SEO:

### Плагины:
1. **Yoast SEO** или **Rank Math** — для оптимизации
2. **WP Super Cache** — для скорости
3. **Contact Form 7** или **WPForms** — для форм

### Настройки:
- Заголовок страницы: "Откачка септиков в Олекминске | Ассенизатор 24/7"
- Описание (meta description): "Срочная откачка септиков и выгребных ям в Олекминске. Цены от 2000₽. Работаем круглосуточно. Звоните!"
- URL: `ваш-сайт.ru/otkachka-septikov-olekminsk`

## Форма обратной связи:

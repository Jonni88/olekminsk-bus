# Упрощённая система BusTime

## Архитектура

```
Водитель → Telegram бот → Backend → Android приложение
                ↓              ↓
           Отметка на      API запрос
           остановке       положения
```

## Компоненты

### 1. Telegram бот для водителей
- Кнопка "Отправился с остановки"
- Кнопка "Прибыл на остановку"
- Выбор маршрута
- Автоматическая отправка GPS

### 2. Backend (Python FastAPI)
- Принимает данные от бота
- Хранит последнее положение автобусов
- API для Android приложения
- Хостинг: ~500₽/мес (или бесплатно на Railway/Render)

### 3. Android приложение (доработка текущего)
- Показывает статус: "Едет к остановке X" / "На остановке Y"
- Примерное время прибытия
- Карта с маршрутом (без real-time движения)

## Данные для расчёта

Примерное положение считаем по:
- Времени отправления с предыдущей остановки
- Средней скорости на участке
- Расстоянию между остановками

## MVP (минимум)

Первая версия без backend:
- Водители пишут в Telegram-группу: "М1 выехал с автовокзала 10:05"
- Ты вручную обновляешь Google Sheets
- Приложение показывает статус из таблицы

**Стоимость MVP: 0₽ (только твоё время)**

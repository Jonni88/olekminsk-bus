# 2026-02-06 — Работа над приложением "Автобусы Олёкминска"

## Что было сделано

### Создано нативное Android-приложение
- **Проект:** `olekminsk-bus-native`
- **Технологии:** Kotlin, Android SDK 35 (для совместимости с Android 16)
- **Функции:**
  - Отображение расписания автобусов
  - Рекламный баннер внизу экрана
  - Монетизация через местную рекламу

### Проблемы и решения
- **Проблема:** Приложение падало на Android 16
- **Причина:** WebView несовместим с Android 16 (API 36)
- **Решение:** Переход на 100% нативный UI (TextView, CardView, ScrollView) без WebView

### Монетизация (спланировано)
- **Модель:** B2B + местная реклама
- **Тарифы:**
  - Баннер в приложении: 3000₽/мес
  - White-label для других городов: 50 000₽
  - Поддержка: 5000₽/мес

### Google Sheets интеграция
- Планировалась загрузка данных из Google Sheets
- Структура: листы "Расписание", "Реклама", "Объявления"
- Обновление без пересборки APK
